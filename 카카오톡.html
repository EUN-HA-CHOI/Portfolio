Kakao React Web App - Kakao App Kakao App

kakao web app은 SPA react.js를 사용하여 만든 앱 입니다.

html,css 먼저 만들고 css는 개발 효율을 위해 scss로 변환하여 react.js로 구현하였습니다.

설명란
주요기능 구글 계정,SNS 로그인 인증구현
Device Mobile
Frontend react,axios,react-router-dom,react-icons,react-dom,prop-types,node-sass,gh-pages,firebase,classnames
나의 역할
  • firebase를 사용해 실시간 데이터 베이스를 연결하여 로그인 인증구현
  • axios를 통해 비동기적으로 API를 호출
  • scss를 사용하여 유지보수와 개발의 효율을 높임
  • prop-type 을 이용해 prop으로 넘어오는 입력값들이 어떤 타입을 가져야하는지 정의함
  • react-router-dom을 사용하여 SPA 구현
  • github pages 를 사용하여 배포
